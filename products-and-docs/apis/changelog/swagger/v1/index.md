- ### Swagger API [v1]
  + ##### General model
    * Multiple sign-based sort order. No sign - ascending, minus sign - descending. Example: 'sort=displayName,-status'
    * Adding the new status **"onsale"** and **"offsale"** instead of **"active"**.
